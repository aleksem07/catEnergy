Команда npm test запускает тест editorconfig и stylelint:

    editorconfig проверяет отступы, концы строк, пустые строки во всех файлах;
    stylelint проверяет ошибки препроцессорных файлов по кодгайду Академии.

npm start запускает browsersync и следит за изменениями стилей и вёрстки;
npm run build запускает компиляцию выбранного препроцессора.
